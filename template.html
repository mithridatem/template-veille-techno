Fiche de veille

Fail2Ban

Outil léger de prévention des intrusions qui protège les systèmes Linux contre les attaques

Thème :

Date :

Auteur / Source :

Description :

Il surveille les fichiers journaux pour détecter les tentatives répétées de connexion infructueuse et interdit temporairement les adresses IP en infraction à l'aide de règles de pare-feu. Il est largement déployé pour sécuriser des services tels que SSH, les serveurs de messagerie et les applications web avec une configuration minimale.

Surveillance des fichiers journaux : Surveille les journaux pour détecter les échecs répétés des tentatives d'ouverture de session. Bannissement automatique des IP : Bloque dynamiquement les IP suspectes via iptables, firewalld ou d'autres pare-feu Linux. Filtres spécifiques aux services : Filtres prédéfinis pour SSH, Apache, NGINX, etc. Politiques d'interdiction personnalisables : Définissez des seuils, des durées de bannissement et des règles de liste blanche/noire.
